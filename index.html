Productividad y diversión al máximo

Eleve su productividad y concentración con la vista multitarea, el modo minipantalla y la función No molestar. O elimine toda notificación del trabajo mientras ve vídeos o juega.

Vista multitarea

Las áreas de trabajo le ayudan a organizar sus pendientes. Mantenga el trabajo y el ocio separados, pero accesibles con solo una pulsación.

Modo minipantalla

Trátese de una película, un juego o un proceso del terminal, el modo minipantalla le permite estar al tanto de una cosa mientras trabaja en otra.

No molestar

Apártelo todo y conserve su concentración en el trabajo, o mantenga a raya todas las notificaciones mientras mira una película. El modo No molestar interrumpe la emisión de notificaciones.

Música

Organice y escuche su música. Navegue por los álbumes, use la búsqueda rápida y cree listas de reproducción.

Epiphany

Navegue por internet con un navegador rápido y ligero. Epiphany le permite visitar páginas y aplicaciones web modernas al tiempo que consume poca batería.

Correo

Gestione múltiples cuentas de correo electrónico de forma rápida con mensajes de correo electrónico organizados en conversaciones, búsqueda instantánea, notificaciones y mucho más.

Fotos

Importe, organice y edite fotografías. Cree un pase de diapositivas. Compártalas en Facebook o Flickr.

Vídeos

Reproducción inteligente y sencilla de películas con videoteca, previsualizaciones de fotogramas en la barra de búsqueda, compatibilidad con subtítulos, pantalla completa inteligente y reanudación a partir del último punto de reproducción.

Calendario

Cree y visualice eventos fácilmente. Sincronícelos con cuentas en línea como las de Google.

Archivos

La barra de direcciones inteligente hace más fácil la navegación web, las búsquedas o el autocompletado de direcciones. Navegue rápidamente gracias a la vista por columnas y disfrute de la navegación avanzada con herramientas como el historial de pestañas.

Terminal

Esquemas de color intercambiables diseñados para evitar la fatiga visual, historial de pestañas, nomenclatura inteligente, notificaciones de finalización de tareas, copia y pega integrado, búsqueda de tareas pendientes, protección contra el pegado y mucho más. ¿Acaso no se puede enseñar trucos nuevos a una antigua aplicación ?

Código

Hecho a medida con guardado automático, carpetas de proyectos, integración de Git, múltiples paneles, espacios en blanco inteligentes, soporte para EditorConfig, Mini Map, símbolos de Vala y extensiones como Terminal, vista previa web y emulación Vim. Con Código no volverá a necesitar otro editor.

Controles de los padres

Tiempo ante pantalla

Establezca límites de tiempo según el usuario para los días de la semana, fines de semana o ambos.

Uso de internet

Gestione las páginas web permitidas. Las reglas afectan a todas las aplicaciones para el usuario, incluso si se utiliza un navegador web diferente.

Gestione las aplicaciones

Elija qué aplicaciones son seguras para que accedan los más pequeños. Opcionalmente también puede permitir el acceso con su contraseña.

Todo lo que hacemos es de código abierto

Nuestra plataforma consiste totalmente de componentes de código abierto y se basa en una plataforma sólida de programas libres (como GNU/Linux). Además, colaboramos de manera activa con el ecosistema para mejorarlo para todos.

Seguro y respetuoso de la privacidad

Como el código fuente está disponible para su revisión, cualquiera (un investigador de seguridad, un usuario preocupado o un fabricante que incluye el sistema en sus equipos) puede verificar que el software es seguro y no filtra información personal.

Avisos de seguridad

Hecho para los desarrolladores

Tiene acceso completo a nuestro código fuente por si quiere añadir una funcionalidad para complementar su aplicación o si tiene curiosidad por ver cómo funciona una prestación o una pauta de diseño. Cópielo, aprenda de él, remézclelo, modifíquelo y redistribúyalo.

Participar

Fáciles de usar. Ideales para un uso eficiente del teclado.

elementary OS está diseñado para que sea fácil de entender y de usar para el nuevo usuario. Esto no quiere decir que sea limitado; los potentes y personalizables atajos de teclado garantizan mantenerse productivo, ya sea el primer o el enésimo día que usa elementary.

Respeto por la privacidad. En todos los ámbitos.

Sus datos le pertenecen exclusivamente a usted. No hacemos ofertas de publicidad ni recopilamos datos personales. Nos financiamos directamente gracias a nuestros usuarios, que pagan lo que desean por elementary OS y por las aplicaciones del Centro de aplicaciones. Así es como se debe hacer.

Nuestra normativa de privacidad

Sin escuchas

elementary OS le a ayuda a controlar el funcionamiento de las aplicaciones. Cuando una aplicación use el micrófono, se muestra un indicador para informar acerca de ello. Cuando una aplicación utilice demasiada energía, el indicador de energía lo marcará.

Servicios de ubicación

Cuando una aplicación quiera acceder a su ubicación, tiene que pedirlo. Se muestra un mensaje que indica qué aplicación y la precisión que solicita. Siempre se puede revocar el acceso en Configuración del sistema ▸ Seguridad y privacidad.

Mantenimiento

elementary OS puede mantener en orden los archivos temporales y los eliminados. Esto no solo conserva libre el almacenamiento de su ordenador, sino que ayuda a garantizar que los datos privados no vuelvan a ser molestos.
